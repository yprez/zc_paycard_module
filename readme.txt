Payment module for Paycard (Israel)
(http://paycard.co.il/)

v1.0, 20/04/2011

Includes English and Hebrew language files.

Created by: Yuri Prezument (@yprez)

This module is free software realased under the terms of the GNU

Installation:
	1. Upload the includes folder to appropriate directories.
	2. Go to Paycard's -> Create a 'payment button' -> Mail link, and copy the numer from "MerchantID=???"
	3. Go to to your admin panel, install the module, use the merchant id from step 2.
	
For a working example see:
	http://otakushop.co.il/
